基本方法

¶普通文本、一级~四级标题

Ctrl + 0 到 Ctrl + 4

¶加粗

Ctrl + B

¶斜体

Ctrl + I

¶下划线

Ctrl + U

¶删除线

Shift + Alt + 5

¶行内代码块

Shift + Ctrl + ` 或 `行内代码块` (其中 ` 指键盘Tab键上面的那个键)

¶超链接

Ctrl + k 或 [超链接-跳转到百度](www.baidu.com)

超链接-跳转到百度

¶表格

Ctrl + T (支持拖拽移动、网页端表格复制转换)

内容 数据
表格测试 100

¶引用

Ctrl + Shift + Q 或 > (引用中可再插入引用,按Enter键结束引用)

¶插入图片

Ctrl + Shift + I (支持本地或网络上的图片)

¶独立公式块

Shift + Ctrl + M 或 $$内容$$ (混编公式块用$内容$)
$$
log_2n
$$

¶部分功能需要在配置文件中开启才能使用

¶代码块

先输入` ` ` 然后Enter键

¶无序列表

输入 - 或 + 或 * 再输入空格 (列表中按Tab键插入子列表)

  • 无序列表1

¶有序列表

输入1.即可再加空格即可，类推 (列表中按Tab键插入子列表)

  1. 有序列表1
  2. 有序列表2

¶任务列表

- [ ] 内容

  • [ ] 任务列表

¶脚注

[^脚注编号] + [^ 脚注编号]:教注内容

X2

H2O
